#### About Data_generator:

​	为了避免数据传输麻烦，部分题目已准备数据生成器构造数据，请进入相关题目data目录下使用一下命令生成题目：

```bash
make data
```

​	对于脚本细节，请查看Makefile文件

​	支持数据生成器题目列表：

- route
- ba_king
- route+
